# E-Commerce Website

## Project Overview

A Full Stack E-Commerce Website developed using React JS, Node.js, Express.js, and MongoDB.

## Technologies Used

### Frontend

* React JS
* React Router DOM
* CSS

### Backend

* Node.js
* Express.js

### Database

* MongoDB Atlas

## Features

### Authentication

* Register
* Login
* Forgot Password

### Authorization

* JWT Authentication
* Role-Based Access Control (RBAC)

### Product Management

* Add Product
* Update Product
* Delete Product
* View Products

### User Features

* Product Search
* Product Filter
* Product Sorting
* Shopping Cart
* Wishlist
* Checkout
* Order History

### Creator Features

* Creator Dashboard
* Order Management
* Order Tracking

## How To Run

### Backend

cd ecommerce-backend

npm install

node server.js

### Frontend

cd ecommerce-frontend

npm install

npm run dev
